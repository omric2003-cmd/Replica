Parsing rules
  • Speaker labels — standalone UPPERCASE name lines start a dialogue row (incl. grouped names like CHLOE/EMMA).
  • Wrapped dialogue — following non-speaker lines are joined until the next speaker or a blank line.
  • Scene headers — lines such as "1 EXT. GARDEN - DAY" are ignored.
  • Description / action — prose paragraphs without a speaker label are ignored.
  • Multi-speaker — grouped names stay in one source cell, but each name is counted separately.
  • Review flags — ambiguous rows are marked suspicious for you to confirm.
